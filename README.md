[Via 轻插件](https://viayoo.com/zh-cn/)
======

`Via 轻插件`是专门为 Via 浏览器定制的 JavaScript 插件

Via 浏览器如何加载插件
------
当页面加载完成，即 dcoument.readyState 的值为 complete 时，浏览器向网页注入相应的脚本
